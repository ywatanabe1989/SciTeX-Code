# SciTeX MCP Servers

Model Context Protocol (MCP) servers for SciTeX modules, enabling bidirectional translation between standard Python and SciTeX format.

## Quick Start

### Install All Servers
```bash
./install_all.sh
```

### Configure MCP
See `mcp_config_example.json` for Claude Desktop configuration.

### Launch All Servers
```bash
./launch_all.sh
```
