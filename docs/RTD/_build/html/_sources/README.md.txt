# SciTeX Read the Docs

This directory contains the Read the Docs documentation for SciTeX.

## Structure

- `conf.py` - Sphinx configuration
- `index.rst` - Main documentation index
- `api/` - API reference documentation
- `modules/` - Module-specific documentation
- `requirements.txt` - Documentation dependencies
- `Makefile` - Build scripts

## Building Documentation

```bash
cd docs/RTD
make html
```

## Read the Docs Integration

This directory is configured for automatic Read the Docs builds.

## Contents

- **Installation Guide** - Getting started with SciTeX
- **API Reference** - Complete module documentation  
- **Core Concepts** - Architecture and design principles
- **Quickstart** - Fast introduction to key features
- **Module Guides** - Detailed guides for each SciTeX module

The documentation is automatically generated from docstrings and includes comprehensive examples for all 30+ SciTeX modules.
